Research Projects

2024 - Present

Phénotyp

Phenotyp is a project focused on the development of an automated system to assess criteria of the Tahitian Pearl. This project aims to enhance the evaluation process of pearls by utilizing advanced image processing and deep learning/machine learning techniques.

Objectives: Develop scalable, automated pipeline for pearl shape classification using modern computer vision and machine learning approaches to reduce expert workload and improve consistency.

Collaborators: Direction des ressources marines (DRM), Pearl Quality Control Cell (CCQP), Pearl farmers

Funding: DRM & UPF

TROPICAL

Development and annotation of a specialized sentiment analysis dataset focused on French Polynesian tourism. This corpus comprises tourists' comments in French concerning their experiences in French Polynesia.

Results: Annotated tourist comments dataset in terms of sentiment, using the triplet ASOTE extraction method (1,592 comments and 10,729 sentiment-annotated triplets).

Technologies used: Natural Language Processing, Sentiment Analysis, French NLP
